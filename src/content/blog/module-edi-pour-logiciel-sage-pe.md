Arc Gestion developpe un module EDI pour les logiciels Sage Petites Entreprises : Apinégoce, Apimécanique, Apibâtiment. La société Arc Gestion connait bien les bases de données des logiciels Sage Petites Entreprises. Plusieurs besoins sont frequemment exprimés par nos clients: extraire des données pour tout un tas de raison dont les échanges avec les sites internet. Ce module EDI: échange de données informatisées permet d’extraire des données de bases de Access .mdb. Les logiciels SAGE PE fonctionnent une base .mdb. L’application a un principe simple: extraire dans un fichier texte avec séparateur (type CSV) des données issues des bases de données Sage. L’executable peut être lancé manuellement ou automatiquement via une tache planifiée (pas de messages affichés). La 1ère version de cette application sera disponible avec l’extraction d’un fichier issu d’une requête SQL. Les données pourront ainsi être traitées par votre site internet, votre CRM,… La seconde version, qui sera disponible dans les prochaines semaines, permettra d’extraire autant de fichiers que de requêtes SQL. Ainsi, vous pourrez extraire de votre base de données: vos clients, vos articles, vos commandes,…. Nous travaillons déjà sur le prototype de la 3ème version qui permettra, en plus, d’intégrer des données en devis. Ainsi, vous pourrez extraire vos commandes de vos sites internet ou de logiciels tiers et les envoyer dans votre gestion commerciale Apinégoce (dans un 1er temps) sous forme de devis (pas de gestion de stock). Cette 3ème version sera compatible avec la version Apinégoce Evolution V14. Le module EDI officiel de Apisoft a disparu du catalogue en V6. En 2009/2010, Arc Gestion avait déjà travaillé sur un prototype de module EDI. L’arrivée d’un développeur dans l’équipe de Arc Gestion permet de mettre en oeuvre les solutions qui avaient été trouvées. L’extraction est programmée en Python.
