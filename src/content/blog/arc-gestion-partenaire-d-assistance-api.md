Arc Gestion est partenaire du site Assistance API. Inscrivez-vous gratuitement sur le forum d’entraide Assistance-API et obtenez de l’aide gratuite sur les logiciels Sage Apibâtiment Batigest, Apimécanique Turbo Gestion, Apinégoce Apisoft, Apicommerce de détail et Comptabilité. Une communauté présente vous pourra vous répondre ainsi qu’une équipe de téchniciens Sage. Apifrog, mascotte du site d’assistance Sage, vous présente le site: Inscrivez dès maintenant en 30 secondes Postez votre message d’erreur Réalisez les manipulations décrites par les techniciens Votre problème est réglé gratuitement NB: pour les problèmes techniques trop complexes, Assistance API ne pourra pas vous aider. Dans ce cas, contacter votre revendeur ou appelez Assistance API au 0.892.490.172.
