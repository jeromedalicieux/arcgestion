Sage avait annoncé l'arrivée de la mobilité dans les logiciels Sage Petites Entreprises.

## Sage Online Access

Le blog Assistance-Api.fr en dit plus sur la solution **Sage Online Access**.

## Les fonctionnalités

Sage Online Access permet de :
- Accéder à votre logiciel via un navigateur web
- Travailler à distance
- Consulter vos données en mobilité

## Les avantages

- **Accessibilité** : De n'importe où, n'importe quand
- **Simplicité** : Aucune installation sur le poste client
- **Sécurité** : Données centralisées et sécurisées

## Compatibilité

Sage Online Access est compatible avec :
- Sage Batigest i7
- Sage Apinégoce i7
