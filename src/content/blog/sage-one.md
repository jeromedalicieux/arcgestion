Début 2012, Sage décide de lancé sa nouvelle offre Sage One au Royaume-Uni, aux Etats-Unis et en Irlande. C’est une application en mode cloud pour les petites et moyennes entreprises. Sage One est une solution de gestion comptable sur le Web. Qu’est ce que le « cloud computing » ou « informatique dans le nuage »? Le « cloud computing » est une sauvegarde de données et de traitements informatiques sur des serveurs distants. Grâce au cloud computing, vous pouvez avoir accès à des informations enregistré dans le « nuage » en libre-service, c’est à dire: à n’importe quel moment, n’importe où et sur n’importe quelle plateforme (pc, pc portable, tablette, mobile…). pour plus d’information : cliquez ici Sage One utilise le SaaS Le Cloud computing permet l’accès à des données dans le nuage, mais pourquoi Sage se lance dans le cloud computing. Sage utilise le mode SaaS (software as a service). Le SaaS est un mode d’utilisation d’une solution logicielle, que l’on utilise à distance par le biais du web. Le mode SaaS permet d’utiliser son logiciel de gestion commerciale lorsque vous êtes en situation de mobilité. L’autre avantage est que vous n’avez pas de contrainte de coût pour l’installation, la mise à jour et la maintenance technique, puisque tout se fait par le web. Les fonctionalités de Sage One votre projet, vos comptes disponible n’importe où, n’importe quand, sur le web gestion clients et fournisseurs gestion de la trésorerie statistique sur les performances de la société facturation suivi du temps soumettre la TVA en ligne visualiser les tâches en cours Avertissement lors de factures impayées enregistrement des transactions bancaires La vidéo de Sage One disponible sur youtube Sage One n’est pas encore disponible en France.
