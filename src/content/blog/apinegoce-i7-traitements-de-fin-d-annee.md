La direction générale des finances publiques (DGFIP) exige d’imprimer les factures au format PDF dans le cadre des contrôles de comptabilité informatisée. Il n’existe pas de cloture dans une gestion commerciale comme cela existe en comptabilité. Il convient donc d’archiver les données via une procédure intégrée dans le logiciel : « Utilitaires / Archivage des données / Lancement ». Dans l’archivage, nous trouverons: – Des fichiers PDF avec les factures clients et fournisseurs – Une sauvegarde de la base de votre logiciel – Un fichier XML de présentation des informations Tous ces éléments sont intégrés dans un fichier .zip. Le fichier est protégé par un mot de passe. Il est également nécessaire d’éditer un inventaire. Cette édition est disponible : « Stocks / Edition d’inventaire« .
