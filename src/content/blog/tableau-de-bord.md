Les dirigeants d'entreprises ont besoin de suivre leur activité.

## L'importance du tableau de bord

Un bon tableau de bord permet de :
- Suivre l'activité en temps réel
- Identifier les problèmes rapidement
- Prendre les bonnes décisions

## Les limites des données comptables

Les données comptables sont :
- Historiques (passé)
- Pas toujours pertinentes pour le pilotage
- Disponibles avec retard

## Les indicateurs clés (KPI)

Un bon tableau de bord suit les indicateurs pertinents :
- Chiffre d'affaires
- Marges
- Encours clients
- Activité commerciale

## Les solutions

Les logiciels de gestion comme Codial ou Sage intègrent des tableaux de bord.
