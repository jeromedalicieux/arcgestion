Issu d’un partenariat technique avec ETAI, Arc Gestion a développé une application entrant dans le processus d’échanges de données de www.JeGereMaVoiture.com. Arc Gestion en tant que distributeur des logiciels Apimécanique et distributeur agréé Script par Sage PE a développé une application d’extraction de données. Via une application Windows développée par la société Arc Gestion, les garages Peugeot pourront extraire très simplement et très rapidement les informations nécessaires à alimenter la base de données du site www.JeGereMaVoiture.com. La révolution pour les garages Peugeot réside dans plusieurs points: Réduction du nombre de manipulation pour extraire les données. Il était nécessaire d’extraire 5 fichiers depuis Apimécanique. Donc au moins 5 manipulations et 5 risques d’erreurs. Réduction du temps d’extraction. En passant de plusieurs heures à moins de 5 minutes pour les plus grosses bases, l’extraction est beaucoup plus rapide. Extraction de données même si l’application est utilisée par d’autres personnes. Extraction de données même si Apimécanique n’est pas installé sur l’ordinateur. Cette application est compatible avec toutes les versions de Apimécanique jusqu’à 6R3 et tous les environnements Windows (XP, Vista, Windows 7 et Windows 8). L’application sera distribuée par le groupe ETAI sous sa marque. Arc Gestion est en mesure de développer d’autres applications spécifiques d’extraction de données des logiciels Sage Petites Entreprises (ou d’autres éditeurs) sur étude d’un cahier des charges.
