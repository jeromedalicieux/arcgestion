**Windows 8** est la dernière édition du système d'exploitation de Microsoft.

## Compatibilité Sage

Les logiciels Sage PE version 11 et supérieures sont compatibles avec Windows 8.

## Versions antérieures

Les versions antérieures à la version 11 nécessitent une mise à jour pour fonctionner sur Windows 8.

## Notre recommandation

Avant de migrer vers Windows 8, vérifiez la compatibilité de vos logiciels et faites les mises à jour nécessaires.

## Arc Gestion vous accompagne

Nous pouvons vous aider à :
- Vérifier la compatibilité
- Mettre à jour vos logiciels
- Migrer vers Windows 8

Contactez-nous avant de migrer !
