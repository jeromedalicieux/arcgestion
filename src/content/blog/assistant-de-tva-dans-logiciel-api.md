Sage a mis en place un assistant de TVA dans ses versions 12 de ses logiciels.

## L'assistant TVA

L'assistant TVA automatise :
- L'application des taux
- Les alertes de non-conformité
- Les modifications de comptes

## Les fonctionnalités

### Application automatique
Les nouveaux taux sont appliqués automatiquement aux bonnes dates.

### Alertes
Le logiciel vous alerte en cas de non-conformité.

### Comptes comptables
Les comptes sont mis à jour automatiquement.

## Versions concernées

L'assistant TVA est disponible uniquement sur les versions 12 des logiciels :
- Sage Batigest V12
- Sage Apinégoce V12
- Sage Apimécanique V12

Si vous n'êtes pas en version 12, contactez-nous pour une mise à jour.
