Sage Online Access est disponible dans les versions 1.1 de Batigest i7 et Apinégoce i7.

## La mobilité pour votre entreprise

Avec Sage Online Access, vous pouvez accéder à votre logiciel en utilisant n'importe quel navigateur web.

## Comment ça fonctionne ?

1. **Installation** : Un serveur doit être configuré
2. **Connexion** : Accédez via votre navigateur
3. **Utilisation** : Travaillez comme si vous étiez au bureau

## Les avantages

- **Mobilité** : Travaillez de n'importe où
- **Flexibilité** : Utilisez n'importe quel appareil
- **Sécurité** : Connexion sécurisée

## Les prérequis

- Version i7 1.1 minimum
- Serveur correctement configuré
- Connexion internet stable
